# SQL Project - Online Store Management System

A complete SQL-based database system to manage operations of an online e-commerce store. This project includes database schema, sample data, and queries for day-to-day store management.

Features:
- Customer management with details and order history
- Product inventory tracking with categories and stock levels  
- Order processing and invoice generation
- Sales reports and analytics queries
- Relationships between customers, products, and orders using foreign keys

Tech Used:
- MySQL / SQL Server
- SQL DDL & DML queries

How to Run:
1. Clone/download this repo
2. Open MySQL Workbench or any SQL client
3. Run the Online Store Management.sql file to create tables and insert sample data
4. Execute the queries at the bottom of the file to test reports

Database Tables:
- Customers - Stores customer info
- Products - Product catalog with price & stock
- Orders - Order details with customer reference
- OrderDetails - Products in each order
- Categories - Product categories

Sample Queries Included
1. Find total sales per month
2. List top 5 best-selling products  
3. Get customers with most orders
4. Check low stock products
